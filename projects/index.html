Trentham United Harriers

Twig, SCSS, Gulp, GravCMS, Javascript, HTML

Built for a local running club to upgrade their website to something that would be easier for their Web Admins to use when updating content, while still being flexible and powerful enough to extend when needed. This was a full overhaul project with a new responsive design, HTTPS and full content/results being carried over from their old website. Built utilizing GravCMS for it's ease of use and editing, while still being powerful enough to extend when needed. I built a fully custom theme (GravCMS utilizes Twig templating for themes) and used Gulp to manage SCSS compilation and CSS prefixing for maximizing browser support.
